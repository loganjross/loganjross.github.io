Slide

A user-focused, effortless web3 wallet.

I came into Slide with the purpose of cleaning up the frontend codebase and revamping the UI/UX for the embedded wallet widget. We've made an incredibly smooth and easy to use web3 user experience, all available from a simple email-based login. One of the major frictions Slide solves is allowing easy on and off ramping from fiat to cryptocurrencies, as well as the NFT Checkout feature allowing a user to purchase an NFT, direct from a marketplace with their debit/credit card. many.exchange

A composable UI for the Phoenix DEX on Solana.

After the Solana DeFi ecosystem lost its backbone in the Serum DEX, the community needed not only a new exchange to build upon but the tools and example of how to do so. We built a composable, easy to use swap interface and developer SDK for the Phoenix DEX that provids developers a way to quickly build upon the Phoenix DEX and restore the Solana DeFi ecosystem to a healthier state. Jet Protocol

A DeFi protocol built on the Solana blockchain.

As the first employee of Jet, I was tasked with creating our user interface. I wanted the product to match our mission within DeFi as a whole—to disrupt the status quo. We created something not only unique, but also powerful and intuitive to use. Considering the end goal of decentralization, we wanted to ensure the development of the protocol and interface well into the future. This lead us to create a TypeScript library for easy integration with and use of our on-chain programs, along with a stylesheet package to perpetuate our unique designs.
